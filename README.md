## 기능 요구사항

* 체스판 초기화
    * 기물의 기본 위치에 맞게 기물을 생성한다.
    * row는 1~8, column은 a~h로 표현한다.
    * 체스판에서 각 진영은 검은색(대문자)과 흰색(소문자) 편으로 구분한다.

* 기물의 종류
    * 폰(Pawn)
    * 비숍(Bishop)
    * 나이트(Knight)
    * 룩(Rook)
    * 퀸(Queen)
    * 킹(King)
